# Project Traffic Light Simulator 2 Arah Arduino Uno
Project ini adalah **simulasi lampu lalu lintas 2 arah** menggunakan **Arduino Uno** dan **LED Traffic Light**.  
Didesain khusus untuk **edukasi** agar mudah memahami cara kerja lampu merah, kuning, dan hijau di persimpangan jalan.

## **🎯 Tujuan Project :**
- Memberikan media belajar sederhana tentang **sistem lalu lintas**.
- Melatih pemahaman dasar **Arduino programming**.
- Menghubungkan konsep **logika waktu** dengan aplikasi nyata (traffic light).
- Sebagai **kit edukasi IoT / mikrokontroler** untuk pelajar dan pemula.

## **Komponen yang Digunakan:**
- 1x Arduino Uno R3 SMD
- 1x Usb Cable Printer ( Untuk Power Arduino Uno )
- 2x Led Traffic Light ( Red, Yellow, Green )
- 1x Breadboard [ Tidak Wajib ]
- Jumper Cable **[ Secukupnya ]**

## ⚡ Wiring Pin
- Traffic Light 1 → Pin 2, 3, 4  
- Traffic Light 2 → Pin 5, 6, 7  

*(Susunan bisa diubah sesuai kebutuhan di kode program)*  

## 📖 Cara Menggunakan
1. Rakit rangkaian sesuai wiring.  
2. Hubungkan Arduino ke power (USB atau adaptor).  
3. Upload program ke Arduino Uno melalui Arduino IDE.  
4. Amati simulasi pergantian lampu lalu lintas.
